# 景区路径规划系统

## 1. 项目概述

景区路径规划系统是一个基于 C 语言开发的控制台应用程序，旨在帮助景区管理人员管理景点信息，并为游客提供路径导航服务。系统采用图的数据结构（邻接表）来存储景区景点和路径信息，支持最短路径查询、导游路线规划等功能。

### 1.2 技术栈

- **开发语言**: C 语言 (C11 标准)
- **编译工具**: GCC (MinGW-w64)
- **数据结构**: 邻接表（无向图）
- **算法**: Dijkstra 最短路径算法

## 4. 核心数据结构

### 4.1 图结构定义

```c
// 边节点结构
typedef struct CNode {
    int index;          // 邻接点在顶点数组中的索引
    int length;         // 边的长度
    struct CNode *next; // 指向下一个边节点
} CNode;

// 顶点结构
typedef struct VNode {
    char data[20];      // 景点名称
    CNode *first;       // 指向第一个边节点
} VNode;

// 图结构（邻接表）
typedef struct {
    VNode roadlist[MAX_SIZE];  // 顶点数组
    int nodenum;               // 顶点数
    int edgnum;                // 边数
} ALGraph;
```

### 4.2 用户结构定义

```c
// 用户角色枚举
typedef enum {
    ADMIN,      // 管理员
    VISITOR     // 游客
} Role;

// 用户结构
typedef struct {
    char username[MAX_USERNAME];
    char password[MAX_PASSWORD];
    Role role;
} User;

// 用户列表结构
typedef struct {
    User users[MAX_USERS];
    int count;
} UserList;
```

## 5. 关键函数说明

### 5.1 图操作函数

| 函数名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `createGraph` | 创建图的邻接表 | `ALGraph *graph` | 无 |
| `printGraph` | 以邻接矩阵形式显示图 | `ALGraph *graph` | 无 |
| `addVertex` | 添加景点并建立边 | `ALGraph *graph`, `char *name`, `char *connectName`, `int length` | 成功返回1，失败返回0 |
| `removeVertex` | 删除景点及其边 | `ALGraph *graph`, `char *name` | 成功返回1，失败返回-1 |
| `saveGraphToFile` | 保存图到文件 | `ALGraph *graph`, `const char *filename` | 成功返回1，失败返回0 |
| `loadGraphFromFile` | 从文件加载图 | `ALGraph *graph`, `const char *filename` | 成功返回1，失败返回0 |
| `freeGraph` | 释放图的内存 | `ALGraph *graph` | 无 |

### 5.2 路径规划函数

| 函数名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `dijkstra` | Dijkstra 最短路径算法 | `ALGraph *graph`, `int start`, `int dist[]`, `int path[]` | 无 |
| `findPath` | 查找两个景点间的最短路径 | `ALGraph *graph`, `char *start`, `char *end` | 无 |
| `createGuideGraph` | 创建导游图（有向图） | `ALGraph *graph`, `ALGraph *guidgraph` | 无 |

### 5.3 用户管理函数

| 函数名 | 功能 | 参数 | 返回值 |
|--------|------|------|--------|
| `initUserList` | 初始化用户列表 | `UserList *list` | 无 |
| `registerUser` | 用户注册 | `UserList *list`, `char *username`, `char *password`, `Role role` | 成功返回1，失败返回0 |
| `loginUser` | 用户登录 | `UserList *list`, `char *username`, `char *password` | 成功返回用户指针，失败返回NULL |
| `viewUsers` | 查看所有用户 | `UserList *list` | 无 |
| `saveUsersToFile` | 保存用户到文件 | `UserList *list`, `const char *filename` | 成功返回1，失败返回0 |
| `loadUsersFromFile` | 从文件加载用户 | `UserList *list`, `const char *filename` | 成功返回1，失败返回0 |

## 7. 数据文件格式

### 7.1 graph.dat（二进制格式）

| 字段 | 类型 | 说明 |
|------|------|------|
| nodenum | int | 顶点数 |
| edgnum | int | 边数 |
| 顶点名称 | char[20] | 每个顶点的名称，共nodenum个 |
| 边数量 | int | 每个顶点的边数 |
| 边索引 | int | 邻接点索引 |
| 边长度 | int | 边的长度 |

### 7.2 users.dat（二进制格式）

| 字段 | 类型 | 说明 |
|------|------|------|
| count | int | 用户数量 |
| username | char[MAX_USERNAME] | 用户名 |
| password | char[MAX_PASSWORD] | 密码 |
| role | int | 角色（0=管理员，1=游客） |

## 8. 注意事项

1. **数据文件位置**: 确保从项目根目录运行程序，数据文件位于根目录下
2. **文件编码**: 程序使用 UTF-8 编码，支持中文显示
3. **内存管理**: 程序在退出时会自动释放所有动态分配的内存
4. **数据备份**: 建议定期备份 `graph.dat` 和 `users.dat` 文件
